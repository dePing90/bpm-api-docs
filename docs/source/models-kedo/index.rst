Описание моделей
================

Контур КЭДО — система кадрового документооборота с сотрудниками.

Workspace — рабочее пространство, в котором может работать организация или холдинг.

User — пользователь системы, имеет доступ к какому-либо пространству.
Это может быть сотрудник какого-то отдела, руководитель или сотрудник кадровой службы

Documents — документы, которые подписывают/согласуют сотрудники, руководители, кадровики.

Routes — маршруты согласований: цепочка действий с документом
