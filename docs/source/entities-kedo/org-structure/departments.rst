Подразделение
=============


Подразделение (Department) — это любой структурный элемент компании: холдинг, организация (юр.лицо), филиал, отдел и т.д.

Аргументы подразделения:

* Идентификатор
* Наименование
* Внешний идентификатор
* Родительское подразделение
* Структура
* Руководитель
