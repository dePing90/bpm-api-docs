Создание пачки¶

Метод API для создания нескольких пользователей

Рассмотрим пример эффективного создания нескольких пользователей за один запрос к API. В Body передается модель UserCreationArgsBatchCreateRequest. Формируем запрос:

curl -X 'POST' \
'https://xcom.kontur.ru/api/v1/testswaggerspace/users/create' \
-H 'accept: application/json' \
-H 'Authorization: Bearer xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx' \
-H 'Content-Type: application/json-patch+json' \
-d '{
"items": [
    {
        "login": "test2@mail.ru",
        "firstName": "имя",
        "middleName": "отчество",
        "lastName": "фамилия",
    },
    {
        "login": "test3@mail.ru",
        "firstName": "имя",
        "middleName": "отчество",
        "lastName": "фамилия",
    }
]
}'

В ответ на успешное создание получим 200 OK:
